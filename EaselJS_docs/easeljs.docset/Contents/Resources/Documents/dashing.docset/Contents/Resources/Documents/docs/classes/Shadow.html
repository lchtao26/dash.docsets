Shadow Class

Defined in: Shadow:41
Module: EaselJS

This class encapsulates the properties required to define a shadow to apply to a DisplayObject via its shadow property.

Example

 myImage.shadow = new createjs.Shadow("#000000", 5, 5, 10);
